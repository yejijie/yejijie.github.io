Word Break

题目描述

Given a string s and a dictionary of words dict, determine if s can be segmented into a space-separated sequence of one or more dictionary words.

For example, given
s = “leetcode”,
dict = [“leet”, “code”].

Return true because “leetcode” can be segmented as “leet code”.

Word Break II

题目描述

Given a string s and a dictionary of words dict, add spaces in s to construct a sentence where each word is a valid dictionary word.

Return all such possible sentences.

For example, given
s = “catsanddog”,
dict = [“cat”, “cats”, “and”, “sand”, “dog”].

A solution is [“cats and dog”, “cat sand dog”].

代码

在上面的DFS方法中的for循环中，我们循环的对象是字典，但是在实际情况中，字典可能非常大，我们应该用字符串s来作为循环对象。

其实在对s.substring进行递归之前，还可以先进行判断，能不能继续递归。

然后修改递归DFS前面的if语句为if(wordDict.contains(word) && containsSuffix(wordDict, s.substring(i)))，在实际应用中字典非常巨大的时候还是有意义的。
